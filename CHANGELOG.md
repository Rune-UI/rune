# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-17

### Added

- Core reactive primitives: `signal()`, `computed()`, `effect()`
- Template-based views: `view()`, `html` tagged template
- DOM mounting with morph-based updates: `mount()`
- Async data: `resource()`
- Web Component bridge: `define()`
- Batch updates: `batch()`
- Untracked reads: `untrack()`
- Lifecycle hooks: `onMount()`, `onCleanup()`
- Client-side router: `route()`, `navigate()`, `params()`
- Global store: `store()`
- Server actions: `server()`
- Agent integration: `agent()`, `tool()`
- Sandbox: `createSandbox()`
- Examples: Counter, Todo
- Full test suite
